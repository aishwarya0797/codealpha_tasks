# Task 2: Stock Portfolio Tracker 

##  Description

This project is a simple Python-based stock portfolio tracker that calculates the total investment value based on predefined stock prices and user-input quantities.

##  Features

* Predefined stock price dictionary
* User input for stock selection and quantity
* Validation for invalid inputs
* Calculates total investment value
* Displays detailed portfolio summary
* Option to save results in `.txt` or `.csv` file

##  Technologies Used

* Python
* Dictionary data structure
* Functions
* File handling (.txt and .csv)

##  How to Run

1. Open terminal in the project folder
2. Run the file:

   ```bash
   python stock_tracker.py
   ```

##  How It Works

* Displays available stocks with prices
* User enters stock names and quantities
* Program calculates:

  * Individual investment per stock
  * Total investment value
* Option to save results to a file

##  File Structure

* stock_tracker.py → Main program

##  Outcome

This project demonstrates the use of dictionaries, loops, functions, and file handling in Python to build a real-world mini application.
