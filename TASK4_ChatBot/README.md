# Task 4: Rule-Based Chatbot 

##  Description

This project is a simple rule-based chatbot developed in Python.
It responds to predefined user inputs using conditional statements.

##  Features

* Handles basic user inputs:

  * "hello"
  * "how are you"
  * "bye"
* Provides predefined responses
* Continuous interaction using a loop
* Exit condition when user types "bye"
* Case-insensitive input handling

##  Technologies Used

* Python
* Functions
* if-elif-else statements
* Loops (while)
* Input/Output handling

##  How to Run

1. Open terminal in the project folder
2. Run the file:

   ```bash
   python chatbot.py
   ```

##  How It Works

* Takes user input from the console
* Processes input using a function
* Matches input with predefined rules
* Returns appropriate response
* Repeats until user enters "bye"

##  File Structure

* chatbot.py → Main chatbot program

##  Outcome

This project demonstrates basic chatbot logic using rule-based conditions, along with core Python concepts like functions, loops, and user interaction.
