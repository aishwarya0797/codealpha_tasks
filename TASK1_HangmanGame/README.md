# Task 1: Hangman Game 

##  Description

This project is a simple command-line based Hangman game implemented in Python.
The player has to guess a hidden word letter by letter within a limited number of attempts.

##  Features

* Random word selection
* User input handling
* Limited attempts for guessing
* Displays progress after each guess
* Win/Lose conditions

##  Technologies Used

* Python
* Random module
* Loops and conditional statements

##  How to Run

1. Open terminal in the project folder
2. Run the file:

   ```bash
   python hangman.py
   ```

##  How to Play

* Guess one letter at a time
* If the letter is correct, it appears in the word
* If incorrect, you lose an attempt
* Game ends when:

  * You guess the word (Win)
  * You run out of attempts (Lose)

##  File Structure

* hangman.py → Main game logic

##  Outcome

This project demonstrates basic Python concepts such as loops, conditionals, string handling, and user interaction.
